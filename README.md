# How to run with TERMUX
1. Update & Upgrade Termux
```
pkg update && pkg upgrade -y
```
2. Install Git
```
pkg install git -y
```
3. install python
```
pkg install python
```
## install ini juga
```
apt install python-pillow
```
## instal ini juga
```
pip install colorama
```
4. Clone this repo
```
git clone https://github.com/xyoruz/memek
```
5. Open the folder
```
cd memek
```
6. Setup
```
bash setup.sh
```
7. buat folder .env di termux
```
nano .env
```
8. Run the script
```
python main.py
```
